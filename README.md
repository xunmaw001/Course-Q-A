# Course-Q-A
这款微信小程序搭建起便捷的课程答疑平台，打破学习的时空壁垒。学生遇到难题，随时通过小程序拍照上传题目，或用文字描述疑问。小程序迅速匹配擅长相关领域的老师，提供一对一解答服务。师生交流不仅限于文字，语音、图片等多元形式，让沟通更顺畅。解答过程中，老师还会给出拓展知识点，帮助学生举一反三。系统保存历史答疑记录，方便学生复习回顾。依托微信生态，让学习难题不过夜，学习之路更轻松 。
